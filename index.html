üìñ Overview

This web scraper is designed to extract all contributions, sessions, and materials from the LLRF2025 workshop hosted at Jefferson Lab. The scraper creates organized folders by type (Oral Presentations, Posters, Sessions) and date, downloading contribution information and attachment files.

Python 3.7+ Indico API Auto Download
